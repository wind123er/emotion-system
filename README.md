# Emotion Engine - 情绪引擎系列

中国大学系列情绪引擎 —— 不同学校 × 学历 × 性别的情绪系统。

## 已发布版本

### 🏥 BMU-FE v1.0 — 北京医科大学女硕士情绪
位于 `bmu-fe/` 目录。
基于规则的情绪系统，覆盖北医女研究生的8种情绪、5种人格特质和清华场地触发事件。

### 🎓 THU-FE v1.0 — 清华大学女研究生情绪引擎
位于 `thu-fe/` 目录。
清华女研究生专属情绪系统，8种情绪维度，包含清华大学学术场景（实验失败/论文接收/课题组会/顶会投稿等）的情绪触发规则。
