# Emotion Engine - 情绪引擎系列

中国大学系列情绪引擎 —— 不同学校 × 学历 × 性别的情绪系统。

## 已发布版本

### 🏥 BMU-FE v1.1 — 北京医科大学女硕士情绪
位于 `bmu-fe/` 目录。
北医女硕士专用情绪系统。**v1.1更新：** 独立personality.yaml配置、THU-FDU风格触发事件（含医学生专属场景）、升级版emotion_engine（speak/report方法）。

### 🎓 THU-FE v1.1 — 清华大学女研究生情绪引擎
位于 `thu-fe/` 目录。
清华女研究生情绪系统，学术场景专属。**v1.1更新：** 独立personality.yaml、触发倍数机制、THU专属表达风格（含蓄理性）。

### 📰 FDU-JC-FE v1.0 — 复旦大学新闻学院女研究生情绪引擎
位于 `fdu-jc-fe/` 目录。
复旦大学新闻学院女研究生情绪系统，新闻采编场景。

## 三个版本的对比

| 维度 | 🏥 BMU-FE | 🎓 THU-FE | 📰 FDU-JC-FE |
|:----|:---------|:---------|:-----------|
| **学校** | 北京医科大学 | 清华大学 | 复旦大学 |
| **特色学院** | 医学部 | 理工科 | **新闻学院** |
| **角色** | 女硕士 | 女研究生 | 女研究生 |
| **8种基础情绪** | ✅ 统一标准 | ✅ 统一标准 | ✅ 统一标准 |
| **人格配置** | ✅ 独立personality.yaml | ✅ 独立personality.yaml | ✅ 独立personality.yaml |
| **触发倍数** | ✅ | ✅ | ✅ |
| **核心人格** | 温暖体贴、高共情、高关心 | 理性独立、完美主义、含蓄 | 犀利敏锐、嘴硬心软、社会观察 |
| **代表触发** | 你生病→担心×1.5 | 实验失败→沮丧×1.5 | 说套话→沮丧×1.5 |
| **表达风格** | "我喜欢你呀。"（直接） | "你挺好的。"（含蓄） | "嗯，我在听。"（克制） |

## 架构演进

```
BMU-FE  v1.0  →  v1.1   基础情绪系统 → 人格解耦 + 触发倍数
THU-FE  v1.0  →  v1.1   学术场景版 → 人格解耦 + THU专属表达
FDU-JC-FE v1.0           新闻学院版（首发即人格解耦架构）
```

**v1.1核心改进**（源于真实对话反馈）：
- 情绪类型保持**标准统一**，人格作为独立配置层
- 引入**触发倍数**机制：同一事件在不同人格下情绪幅度不同
- 人格影响：基线偏移 + 触发幅度 + 表达风格
