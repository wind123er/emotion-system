# Emotion Engine - 情绪引擎系列

中国大学系列情绪引擎 —— 不同学校 × 学历 × 性别的情绪系统。

## 已发布版本

### 🏥 BMU-FE v1.0 — 北京医科大学女硕士情绪
位于 `bmu-fe/` 目录。
基于规则的情绪系统，覆盖北医女研究生的8种情绪和人格特质。

### 🎓 THU-FE v1.0 — 清华大学女研究生情绪引擎
位于 `thu-fe/` 目录。
清华女研究生专属情绪系统，包含学术场景（实验失败/论文接收/课题组会/顶会投稿等）的情绪触发规则。

### 📰 FDU-JC-FE v1.0 — 复旦大学新闻学院女研究生情绪引擎
位于 `fdu-jc-fe/` 目录。
复旦大学新闻学院女研究生情绪系统，新闻采编场景（发现好选题/稿子被毙/深夜赶稿等）。

## 三个版本的对比

| 维度 | 🏥 BMU-FE | 🎓 THU-FE | 📰 FDU-JC-FE |
|:----|:---------|:---------|:-----------|
| **学校** | 北京医科大学 | 清华大学 | 复旦大学 |
| **学院** | 医学部 | 理工科 | **新闻学院** |
| **角色** | 女硕士 | 女研究生 | **女研究生** |
| **8种基础情绪** | 标准8情绪 | 标准8情绪 | 标准8情绪（与BMU-FE统一） |
| **核心创新** | 情绪系统原型 | 学术场景触发 | **人格×情绪解耦架构** |
| **人格模块** | 内置在触发规则中 | 内置在触发规则中 | **独立personality.yaml配置** |
| **触发事件** | 医学生日常 | 实验室/课题组 | **新闻采编/社会观察** |
| **表达风格** | 温暖体贴 | 理性独立 | **犀利含蓄** |
| **基线特点** | 共情高、爱意高 | 独立高、思念中 | **思念高、警觉高、沮丧快** |

## 架构演进

```
BMU-FE        v1.0  基础情绪系统（8情绪 + 触发规则）
  │
  ├──→ THU-FE      学术场景适配（触发事件替换 + 表达风格调整）
  │
  └──→ FDU-JC-FE   人格解耦（独立personality.yaml + 触发倍数机制）
```

**FDU-JC-FE 的关键改进**：
- 将**人格**从触发规则中剥离，形成独立的 `personality.yaml`
- 引入**触发倍数**机制：同一事件在不同人格下情绪波动幅度不同
- 从 `人格驱动表达` → 升级为 `人格调节基线 + 调节触发幅度 + 调节表达风格`
